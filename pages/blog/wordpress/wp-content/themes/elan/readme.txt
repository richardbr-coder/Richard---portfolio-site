== Description ==
Elan is simple, clean & minimal WordPress blog theme. It is responsive and beautifully designed blog theme with super easy settings and lightweight to minimize website loading time. Find more details at https://precisethemes.com/wordpress-theme/elan/ and view demo at https://precisethemes.com/demo/elan-free/

== Frequently Asked Questions ==
= Frequently Asked Questions ==
= Installation =
	1. In your admin panel, go to Appearance -> Themes and click the 'Add New' button.
	2. Type in Elan in the search form and press the 'Enter' key on your keyboard.
	3. Click on the 'Activate' button to use your new theme right away.

= How to install demo data of theme ? =
    1. In your admin panel, go to Appearance -> Plugins and click the 'Add New' button.
	2. Type in One Click Demo Importer in the search form and press the 'Enter' key on your keyboard.
	3. Click on the 'Activate' button.
	4. Now go to Appearance -> Import Demo Data and click the 'Import Demo Data' button.
	
= How can I customize Elan? =
	To customize the Elan theme you have to open the Customizer by accessing the Appearance -> Customizer and customize to taste.
    
= Where can I add widgets? =
    Elan includes widgets areas, located underneath the main content and sidebar. Configure these areas by going to Customizer > Widgets in your Dashboard.

= How to add a menu? =
    Elan comes with one Custom Menu in the header, which can be configured via Customizer at Menus. This menu is hidden by default, and will not be displayed unless a Custom Menu is assigned.
